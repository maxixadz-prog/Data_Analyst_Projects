# Разработка дашборда для данных конференций TED

## Описание проекта
Работа с данными конференций TED. Создал дашборд с основными показателями конференции: динамика выступлений, популярные темы, география участников.

## Навыки и инструменты
Yandex DataLens

## Общий вывод
Дашборд создан, отвечает на поставленным задачам.
