# Разработка дашборда для данных конференций TED

## Описание проекта
Работа с данными конференций TED. Создал дашборд с основными показателями конференции: динамика выступлений, популярные темы, география участников.

## Навыки и инструменты
Yandex DataLens

## Что сделано:
- Подключил данные и подготовил их для дальнейшего анализа.
- Настроил интерактивный дашборд, добавил фильтры по странам, конференциям, тегам выступления, дата записи и выступления
- Настроил графики: топ-10 самых смешных выступлений и топ-20 самых популярных тегов

## Общий вывод
Дашборд создан, отвечает поставленным задачам. Позволяет быстро отслеживать ключевые тренды конференций TED.
