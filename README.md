# Описание проектов

## Проекты выполнял в Яндекс Практикуме

| Название проекта | Описание | Используемые библиотеки / Языки / Инструменты |
|------------------|----------|-----------------------------------------------|
| **Проект №1. SQL. Анализ данных и решение ad hoc задач** | Работа с базой данных онлайн-игры «Секреты Темнолесья». Решил серию аналитических задач с помощью SQL-запросов и сформулировал выводы по их результатам. | SQL, DBeaver, PostgreSQL |
| **Проект №2. DataLens. Разработка дашборда** | Работа с данными конференций TED. Создал дашборд с основными показателями конференции: динамика выступлений, популярные темы, география участников. | Yandex DataLens |
| **Проект №3. Анализ рынка недвижимости** | Анализ данных для агентства недвижимости. Изучил объявления о продаже жилой недвижимости в Санкт-Петербурге и Ленинградской области, выявил самые перспективные сегменты. | Yandex DataLens |
| **Проект №4. Python. Предобработка данных** | Знакомство с данными рынка видеоигр. Проверил корректность данных, выполнил предобработку и подготовил срез для дальнейшего анализа. | Jupyter Notebook, Python, pandas |
| **Проект №5. Исследовательский анализ данных и визуализация на Python** | Анализ рынка общественного питания в Москве. Загрузил и обработал данные, провел исследовательский анализ, визуализировал результаты, сформулировал выводы и рекомендации. | Jupyter Notebook, Python, pandas, matplotlib, seaborn |
| **Проект №6. Python. Анализ данных** | Исследование стартапов. Подготовил датасет, проанализировал динамику и структуру финансирования стартапов, ответил на ключевые вопросы для инвестиционных стратегий. | Jupyter Notebook, Python, pandas, matplotlib, seaborn |
| **Проект №7. DataLens. Анализ бизнес-метрик** | Рассчитал метрики для сервиса доставки еды: DAU, Conversion Rate, средний чек, LTV, Retention Rate. Визуализировал результаты в Yandex DataLens. | Yandex DataLens |
| **Проект №8. Формулировка и проверка гипотез. Статистический анализ данных** | Проанализировал демографию пользователей и особенности использования самокатов. Оценил потенциальную выгоду от распространения платной подписки. | Jupyter Notebook, Python, pandas, matplotlib, seaborn |
| **Проект №9. Анализ результатов A/B-тестирования на Python** | Разработал и провел A/B-тест. Рассчитал параметры, проанализировал результаты и сделал вывод о влиянии изменений. | Jupyter Notebook, Python, pandas, matplotlib, seaborn |
| **Проект №10. Исследовательский анализ данных и проверка гипотез** | Провел исследовательский анализ данных Яндекс Афиши. Сформулировал и проверил гипотезы с использованием статистических тестов. | Jupyter Notebook, Python, pandas, matplotlib, seaborn, proportions_ztest, ttest_ind |
