# Проект: Рынок недвижимости

## Описание проекта
Подготовлен анализ данных для агентства недвижимости. Работа с объявлениями о продаже жилой недвижимости в Санкт‑Петербурге и Ленинградской области.

## Навыки и инструменты
- **DBeaver** — работа с SQL-запросами
- **Yandex DataLens** — визуализация данных
- **Ad hoc задачи** — оперативный анализ под запросы заказчика
- **Аналитическая записка** — подготовка итогового отчёта

## Общий вывод
Интерактивный дашборд создан, отвечает на все вопросы заказчика.
