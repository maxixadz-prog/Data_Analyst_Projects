# Проект: Развлекательное приложение

## Описание проекта
Данные развлекательного приложение с функцией «бесконечной» ленты. В приложении существуют две модели монетизации: первая — ежемесячная платная подписка, которая позволяет пользователям смотреть ленту без рекламы, вторая — демонстрация рекламы для пользователей, которые ещё не оформили подписку.

## Что сделано:
Рассчитаны параметры теста, оценена корректность его проведения и проанализированы результаты эксперимента.

    2) Длительность эксперимента: `1 день` ;
    
    3) Внедрение нового алгоритма рекомендаций не повлияло,результат не принес существенные изменения
    
    4) pvalue=0.5783523649187868 > 0.05. Разница статистически незначима.
