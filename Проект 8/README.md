# Проект: Проверка гипотез в бизнесе

## Описание проекта
Данные крупного сервиса проката самокатов GoFast.Три датасета с данными о пользователях сервиса, их поездках за один год, а также стоимости услуг в подписках. 

## Что сделано
Проанализирована демография пользователей и особенности использования самокатов, а также определение возможной выгоды от распространения платной подписки на самокаты.

## Общий вывод
Выявлены ключевые характеристики распределения данных и определены критические значения.
