# Работа с базой данных онлайн-игры «Секреты Темнолесья»

## Описание проекта
Провожу разведочный анализ данных.Изучаю базу данных в DBeaver.Решаю аналитические задачи,пишу выводы и аналитические комментрарии. 

## Навыки и инструменты
SQL, DBeaver , PostgreSQL
